## Top Menu Callout (above properties)

Add **cssclass** to your note:
> cssclasses: topmenu

Add menu links with this callout:

```python
> [!top|menu] 
> [[Home]] [[Note]] [[Note]] [[Note]] [[Note]]  [[Note]]
```
